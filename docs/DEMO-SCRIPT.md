## Troubleshooting

| Issue | Fix |
|---|---|
| Gallery doesn't load | Hard-refresh (Cmd+Shift+R) · clear cache |
| Scenario crashes at a step | Click "Stop run" in sidebar → re-launch the scenario |
| Charts don't render | Resize browser window slightly to trigger re-render |
| Sidebar collapsed and can't find steps | Click hamburger menu top-left to expand |
| Need to skip to a specific step | Click step name in the sidebar accordion nav |
| AttributionPanel error (engagement.tier1) | Should be fixed — all LEARN steps now use cohort_outcome. If it recurs, stop and re-launch. |
